spa
===

Author: Prasad.A

Enable to get started on building "Single Page Application"
while maintaining the code organized.

Refer: 
* indexhtml.xml to learn about putting runtime files together.

Usage:
* Execute ./package.sh to generate the deliverable index.html

NOTE:
* You need to package whenever (datasource/template) information changes.
* If page was packaged is executed with debug="false", changes to javascript will reflect only on rebuilding.
